API de Chamados - Netcom Telecom

Este projeto é uma API REST feita em Node.js, Express e MongoDB Atlas.

Rotas principais:

  • POST /logar
  • POST /chamados
  • GET /chamados
  • GET /chamados/:id
  • PUT /chamados/:id
  • DELETE /chamados/:id
  • GET /relatorio

Veja a documentação completa no README do GitHub.
